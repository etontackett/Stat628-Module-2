# STAT628 Module 2 Work

This repository contains reproducible analysis scripts for the cranberry heat-stress project.

Confidential course data, derived datasets, and analysis outputs are intentionally excluded from version control.

## Included

- `scripts/clean_yield_raw.py`
- `scripts/check_yield_vs_sensors.py`
- `scripts/lock_analysis_datasets.py`
- `scripts/analyze_yield.py`

## Excluded

- Raw data under `cranberry-data-group6/`
- Derived outputs under `cleaned_data/`
- Assignment PDF

Run the scripts locally in order to regenerate cleaned files and analysis outputs from the confidential source data.
